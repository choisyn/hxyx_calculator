# 幻想英雄 — 道具兑换计算器

这是一个基于浏览器的轻量级道具兑换计算器，用于游戏“幻想英雄”（项目名：Pocket Knight Calculator）。目前实现并公开了一个主要的“活动道具兑换”计算器，后续可扩展为包含更多小工具与计算器。

**主要功能**
- 活动道具兑换计算：支持 S / SS / SSS 道具以及特殊合成项的兑换计算。
- 初始道具设置：可输入已有的 S、SS、SSS 道具和“大活动掉落物”（按 S:SS:SSS = 90:9:1 比例转换）。
- 快捷操作按钮：支持 `0`、预设数量、`+` 增量以及 `Max`（最大兑换）快速填充。
- 结果展示：计算所需体力与兑换后的道具消耗汇总（页面内“计算所需体力”按钮）。

**使用说明**
1. 在浏览器中打开项目根目录下的 [index.html](index.html)。
2. 在“初始道具设置”区域输入你当前拥有的道具数量（S / SS / SSS 或大活动掉落物）。
3. 在相应兑换项中输入想要兑换的数量，或使用快捷按钮快速填充数值。
4. 点击页面下方的“计算所需体力”按钮查看计算结果；使用“重置”清除当前输入，使用“清除缓存”清除本地保存的数据。

**注意事项**
- 页面顶部说明：本计算结果仅供参考。默认假设大活动掉落 S:SS:SSS 比率为 90:9:1，实际游戏掉率可能存在波动，请以游戏实际情况为准。
- 某些兑换项有单项最大兑换限制（界面中以 `data-max` 和按钮提示为准）。

**文件说明**
- [index.html](index.html) — 主页面，包含界面结构与控件。
- [script.js](script.js) — 页面逻辑与计算实现（兑换、快捷按钮、计算与缓存）。
- [styles.css](styles.css), [styles2.css](styles2.css) — 页面样式文件。
- image/ — 兑换项所用图片资源目录。
