# 幻想英雄 · 道具兑换计算器

轻量、可扩展的浏览器端道具兑换工具（Pocket Knight / 幻想英雄）。

当前为单页多视图（SPA）框架，包含“道具兑换（活动）计算器”。

---

主要功能
- 活动道具兑换计算：支持 S / SS / SSS 及特殊合成项
- 初始道具设置：输入已有 S/SS/SSS 或“大活动掉落物”（按 S:SS:SSS = 90:9:1 自动换算）
- 快捷填充：归零、预设数量、增量 (+) 与 Max 快速设置
- 无刷新视图切换：左下角汉堡菜单打开侧栏，切换视图无需刷新

快速开始
1. 在浏览器中打开项目根目录下的 `index.html`
2. 点击左下角菜单打开目录，选择“道具兑换计算器”
3. 在“初始道具设置”填写已持有道具数量；在兑换项中输入目标兑换数量或使用快捷按钮
4. 点击“计算所需体力”查看结果；使用“重置 / 清除缓存”管理输入与本地数据

注意事项
- 结果仅供参考。默认大活动掉落比例为 90:9:1（S:SS:SSS），实际掉率可能波动
- 个别兑换项受 `data-max` 限制，请参考界面提示

项目结构（重要文件）
- `index.html` — 应用框架（侧栏 + 视图容器）
- `script.js` — 框架脚本：侧栏控制、按需加载视图并注入对应脚本
- `styles.css` — 全局样式（包含侧栏与汉堡按钮样式）
- `active/` — 活动道具计算器页面/资源（`index.html`、`script.js`）
- `evolution/` — 进化材料计算器（占位视图）
- `image/` — 图片资源目录

开发与贡献

- 我可以帮你：把 `active/index.html` 改为可注入的片段、增加路由支持或美化界面
